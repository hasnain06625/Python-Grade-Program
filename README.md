# Python-Grade-Prrogram
This is a Python grade calculator

# Overview
This is the Python program calculate a student's grade based on marks and dispays encouraging message.

# How to Run
1. Install Python
2. run : 'grade_calculator.py'

# Features
1. Grade calculator using if-elif-else statements.
2. Input validation (0-100).
3. Function-based implementation.
4. Repeat execution using loop.
5. Execution handling for invalid inputs.
